# STratusBot Cybersecurity Awareness Chatbot Project Part 2

## Project Overview
StratusBot is a cybersecurity awareness chatbot designed to educate users about various aspects of cybersecurity, including best practices, threats, and preventive measures. 
Built using an intuitive interface, STratusBot engages users in conversation, answering questions and providing guidance to enhance their cybersecurity knowledge and practices.
This update overhauls the chatbot and adds GUI elements using WPF. The app is built with .NET 8 and designed for Windows (Visual Studio).

#

## Features
- **Interactive Conversation:** Engage with users through a dynamic chat interface.
- **Knowledge Base:** Access to a wealth of information regarding cybersecurity, including FAQs and best practices.
- **Scenario-Based Learning:** Provide users with real-world scenarios to understand potential cybersecurity threats and responses.
- **Resource Sharing:** Ability to share resources, articles, and further reading materials on cybersecurity topics.
- Keyword-based responses and simple sentiment opener.
- Simple WPF UI with chat bubbles, rounded input, and ASCII header.
